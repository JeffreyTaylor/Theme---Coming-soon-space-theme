A) HTML Structure - top

This theme is a single page template. The intention of this template is to entice users to enter their e-mail, so that you can e-mail them later about updates to your new site!

If you would like to edit the color, font or any elements on the screen, please open the src/css/style.css file.

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

B) CSS Files and Structure - top

I'm only using one CSS file in this theme. This file contains general styling (such as anchor tag colors, font-sizes), as well as the css illustration of the astronaut and moon. At the very bottom of the file, you'll find the media queries necessary to detect and react to screen-size changes that may happen while this page is being displayed.

C) JavaScript - top

This theme imports four Javascript files.

  1. jQuery
  2. jQuery-ui.draggable
  3. Particles.js
  4. Custom scripts.js
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. jQuery.draggable is a Javascript library allows the user to drag things on the screen (the user is able to drag the astronaut around -- how fun!)
  3. Particles.js is the library used to draw and animate the background of the theme.
  4. Custom scripts.js is a custom file written exclusively for this theme. This contains the logic to set up the background as well as logic to validate & submit the subscribe form.
